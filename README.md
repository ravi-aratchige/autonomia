# Autonomia

*Assistive Technology for Motor-Impaired Users*

More details about Autonomia will be added soon!

## Project Setup

Clone the repo.

```sh
git clone https://github.com/ravi-aratchige/autonomia.git
```

```sh
cd autonomia
```

Make sure you have [uv](https://docs.astral.sh/uv/) installed.

Then, install the dependencies:

```sh
uv sync
```

Run the project.

```sh
uv run main.py
```
